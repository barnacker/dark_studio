# Dark Studio Theme for OBS (WIP)
A theme to work in a dark room without getting virtually sunburned from overexposure to the standard "Dark" OBS theme, also greatly reduces impact on your webcam light setup with your secondary screen lighting up your face running regular OBS.

## Requirments
As of now you only need [IosevkaTerm Nerd Font](https://www.nerdfonts.com/font-downloads) to get the same look as you see below.
